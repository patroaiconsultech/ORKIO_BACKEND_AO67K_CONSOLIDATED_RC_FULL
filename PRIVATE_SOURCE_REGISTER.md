# Private Source Register

Release Candidate: OES-RC-0005-R4

This register contains source candidates only. It contains no raw private content.

| Source ID | Type | Status | Raw content included | Direct publication allowed | Content access allowed | Current use |
|---|---|---|---|---|---|---|
| `FOUNDER_CONTEXT_PARTIAL_GDRIVE_001` | Private drive document | Partial / Not ingested | false | false | false | metadata_only_without_content_access |
| `FOUNDER_GPT_ACCOUNT_EXPORT_001` | Potential account export | Not received | false | false | false | none |

## Rule

Adding a source to this register does not authorize its content for use.

`metadata_only_without_content_access` means that the system may record the existence, type and triage need of a source, but may not read, ingest, summarize, index, quote, publish or embed its content.
