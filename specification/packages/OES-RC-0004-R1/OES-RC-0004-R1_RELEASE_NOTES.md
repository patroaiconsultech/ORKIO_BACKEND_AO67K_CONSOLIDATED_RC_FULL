# OES-RC-0004-R1 — Release Notes

OES-RC-0004-R1 introduces OES-007, the Contract and Event Projection layer for ORKIO.

This RC derives command contracts and domain events from the 56 capabilities approved in OES-006. It remains specification-only and is intended to prepare later runtime implementation without changing current behavior.
