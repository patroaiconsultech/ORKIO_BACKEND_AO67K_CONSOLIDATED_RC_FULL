# ORKIO Editorial Board

Este diretório contém os hyperprompts editoriais usados para manter a documentação técnica, executiva e institucional do ORKIO sincronizada com o código.

## Perfis
- `editorial_hyperprompt.md`: prompt principal do Editorial Board.
- `editorial_architecture.md`: documentação de arquitetura.
- `editorial_product.md`: roadmap, produto e release notes.
- `editorial_investors.md`: materiais executivos, valuation e data room.
- `editorial_qa.md`: revisão técnica da documentação.

## Regra operacional
O código e os testes são a fonte primária de verdade. A documentação nunca deve declarar como concluído algo que não esteja comprovado por implementação, smoke test ou release.
