# Experience Core

## Purpose
Experience Core defines how the user perceives Orkio.

It answers: **How does the user feel the intelligence, clarity and trust of Orkio?**

## Components
- Landing
- Onboarding
- Conversation UX
- Documents
- Upload experience
- Evidence view
- Mission timeline
- Executive dashboard
- Voice
- Avatar
- Notifications
- Personalization

## Experience Principle
The user must feel that Orkio helps them understand, decide and build better.
