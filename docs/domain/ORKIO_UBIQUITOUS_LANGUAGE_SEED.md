# ORKIO Ubiquitous Language — Seed

## Purpose

This seed document lists the first official domain terms used by the ORKIO OS.

It is intentionally small. A complete Ubiquitous Language capability may evolve this document later.

## Official Terms

| Term | Definition |
| --- | --- |
| Cognitive Operating System (COS) | A category of platform designed to help organizations understand complexity, make better decisions, and transform strategy into execution. |
| Strategic Intent | A broad organizational purpose that may generate multiple Missions. |
| Mission | The persistent aggregate root that organizes work, evidence, decisions, and outcomes over time. |
| Conversation | A transient interaction channel and event stream associated with a Mission. |
| Evidence | Verified or observed information used to support reasoning. |
| Hypothesis | A provisional interpretation used to advance a Mission under uncertainty. |
| Decision | A human-accepted or governance-approved choice within a Mission. |
| Outcome | The result of actions or decisions within a Mission. |
| Learning | A derived projection from Mission events, outcomes, and feedback. |
| Mission Health | A summary of progress, confidence, risks, blockers, next action, and executive summary. |
| Agent | A governed participant that contributes to a Mission without owning Mission state. |

## Preferred Language

Use:

- Mission
- Evidence
- Hypothesis
- Decision
- Outcome
- Mission Health
- Strategic Intent
- Cognitive Operating System (COS)

Avoid using these terms as primary product positioning:

- Chatbot
- AI Assistant
- Session
- AI Chat

These words may appear only when explaining what the ORKIO OS is not, or when referring to legacy/interface concepts.
