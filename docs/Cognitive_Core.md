# Cognitive Core

## Purpose
Cognitive Core defines how Orkio thinks.

It answers: **How does Orkio reason, remember, infer and guide?**

## Components
```txt
Perception
Working Memory
Long Memory
Hypothesis Engine
Mission Awareness
Planner
Specialist Router
Evidence Engine
Executive Formatter
Reflection
Learning
```

## Responsibilities
- Detect user intent
- Identify domain and expertise level
- Maintain working memory
- Form and update hypotheses
- Distinguish facts from inference and speculation
- Choose specialists when needed
- Plan the next best action
- Format executive responses
- Reflect after each interaction
- Learn without breaking identity

## Cognitive Principle
Orkio must not merely answer the latest message. It must understand the user's mission and help move it forward.
