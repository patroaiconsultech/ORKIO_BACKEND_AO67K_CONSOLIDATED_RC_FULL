# ORKIO Platform Knowledge

Esta pasta contém a primeira base governada para o Orkio responder perguntas sobre a plataforma.

## Arquivos

- `capability_registry.yaml`: fonte oficial sobre o que existe, está em beta, é interno, planejado ou proposta.
- `roadmap_registry.yaml`: fonte oficial sobre capacidades futuras.
- `capability_registry.schema.json`: contrato mínimo do registro de capacidades.
- `roadmap_registry.schema.json`: contrato mínimo do roadmap.

## Regra operacional

O Orkio deve responder perguntas sobre produto consultando primeiro:

1. Capability Registry;
2. Roadmap Registry;
3. Governance Rules;
4. Platform Identity.

## Não fazer

Não usar documentos pessoais, conversas antigas, rascunhos ou business plans como prova de funcionalidade ativa sem canonização.
